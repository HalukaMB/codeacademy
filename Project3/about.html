Discogs Matcher
about
This is a website dedicated to give vinyl diggers, record lovers suggestions which other records to check out that resemble the sound of their favourite artists.
The person behind this little project is a vinyl lover himself. Haluka started collecting records about two years ago and spends a good amount of his free time buying records.
The code is obviously still far from perfect. But Haluka hopes that you have some fun with this project.
"This application uses Discogs' API but is not affiliated with, sponsored or endorsed by Discogs. "Discogs" is a trademark of Zink Media, LLC."
